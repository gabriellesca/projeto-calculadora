## 🎯 Sobre o projeto

Projeto desenvolvido para praticar lógica JavaScript, manipulação do DOM e boas práticas de desenvolvimento front-end. O diferencial deste projeto é a substituição do `eval()` por uma alternativa mais segura usando `Function()`, além do suporte completo ao teclado.

---

## ✨ Funcionalidades

- [x] Operações: soma, subtração, multiplicação, divisão e porcentagem
- [x] Botão apagar caractere (⌫) e limpar tudo (C)
- [x] **Suporte completo ao teclado** — digite sem clicar!
- [x] Exibe a expressão e o resultado separadamente
- [x] Tratamento de erros (divisão inválida, expressão malformada)
- [x] Responsivo para telas menores
- [x] Animação de entrada com CSS puro

---

## ⌨️ Atalhos de teclado

| Tecla | Ação |
|---|---|
| `0–9` | Inserir número |
| `+ - * /` | Operadores |
| `Enter` ou `=` | Calcular |
| `Backspace` | Apagar último caractere |
| `Escape` | Limpar tudo |
| `.` | Decimal |
| `%` | Porcentagem |

## 🧠 Conceitos aplicados

| Conceito | Descrição |
|---|---|
| `Function()` | Alternativa segura ao `eval()` |
| `keydown` | Suporte completo ao teclado |
| `try/catch` | Tratamento de erros de expressão |
| `slice(0, -1)` | Remoção do último caractere |
| CSS Grid | Layout do teclado da calculadora |
| `grid-column: span 2` | Botão 0 ocupando duas colunas |
| CSS Variables | Paleta de cores centralizada |
| `@keyframes` | Animação de entrada |
